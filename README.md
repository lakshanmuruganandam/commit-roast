# 🔥 Commit Roast

**Prepare to be judged based on your terrible git commit history.**

Have you ever written `fix` or `wip` and pushed to main? Your team hates you, and now your terminal does too. 

`commit-roast` aggressively scans your last 30 git commits and brutally roasts you if your commit messages are too short, vague garbage (e.g., "fix", "wip", "update"), or just random keyboard mashes. Finally, it gives you a "Roast Score" and ranks you from "Acceptable" to "Absolute Trash".

## 🚀 Usage

No installation required! Just run it directly using `npx`:

```bash
npx commit-roast
```

Or, install it globally:

```bash
npm install -g commit-roast
```

Then run it anywhere:

```bash
commit-roast
```
